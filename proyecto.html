Módulo Didáctico de Tratamiento de Agua

Conectando la Ingeniería y la Industria 4.0 en Universidad La Salle (ULSA)

De un Módulo Funcional a una Herramienta Didáctica 4.0

Este proyecto rediseña un módulo de tratamiento de agua UV-C existente, integrando un sistema de control y monitoreo remoto IoT (Internet de las Cosas).

El objetivo es transformar este equipo en un módulo didáctico interactivo para que los estudiantes de ingeniería de ULSA puedan experimentar y aplicar de forma práctica los conceptos teóricos de la Industria 4.0.

La Transformación: Funcionalidades IoT

Monitoreo en Tiempo Real

Visualización remota de sensores clave: caudal de agua, estado de la lámpara UV, temperatura y contadores de flujo, todo desde un dashboard web.

Control Remoto

Capacidad de encender o apagar la bomba y la lámpara UV-C a distancia, permitiendo la automatización de ciclos de desinfección.

Plataforma en la Nube

Integración con una plataforma IoT para la recolección, almacenamiento y análisis de datos históricos, abriendo la puerta al Big Data y Mantenimiento Predictivo.

Arquitectura del Sistema

1. Sensores y Actuadores

Miden flujo, T°, etc. y controlan la bomba/UV.

2. Controlador IoT

(Ej. ESP32, Raspberry Pi) Recopila datos y los envía a la nube.

3. Plataforma y Dashboard

App web o móvil para visualizar datos y enviar comandos.

Valor para el Estudiante ULSA

  • Conexión Teoría-Práctica: Aplicar conceptos abstractos de Industria 4.0 (IIoT, Cloud, Big Data) en un sistema físico real.
  • Desarrollo de Habilidades "Full-Stack": Desde la configuración de hardware y sensores hasta la programación del firmware y el desarrollo del dashboard.
  • Experimentación Segura: Un entorno controlado para probar lógicas de automatización y análisis de datos sin riesgos industriales.
